\centerline{ \bf Extra Problems on Limits}
\vskip .3in
\document
\vskip .1in
${\bold 1}$)  Prove that $\, \lim_{n \to
\infty}  \dfrac{2 n + 11}{n + 3} = 2$ using the definition of
convergence. 

\vskip .3in

${\bold 2}$) Prove that $\, \lim_{n \to
\infty}\sqrt{ \dfrac{5 n + 3}{n} } = \sqrt{5}$ using the definition of
convergence.

\vskip .3in

${\bold 3}$) Prove that $a_n= \dfrac{5}{\sqrt{n}} + 3 $ converges 
 using the definition of convergence.

\vskip .3in

${\bold 4}$) Prove that $a_n= \dfrac{ (-3)^n }{4^n + 7} $ 
converges using the definition of convergence.
\vskip .2in 
( note that  $(-3)^n = (-1)^n 3^n $ )
